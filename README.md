# 📊 IPL Data Analysis Dashboard

This repository contains a comprehensive IPL (Indian Premier League) Data Analysis Dashboard built using Microsoft Excel.

## 🎯 Project Overview

The dashboard analyzes IPL match data from 2008–2019 and provides valuable insights into team performance, venue statistics, toss decisions, and match outcomes through interactive visualizations.

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization
- Dashboard Design

## 📈 Key Insights

- Team-wise performance analysis
- Venue-wise match statistics
- Toss win vs match win analysis
- Season-wise trends
- Most successful teams and venues
- Interactive KPI tracking

## 📂 Repository Contents

- IPL Dataset
- Excel Dashboard File
- Dashboard Screenshots
- Project Documentation

## 🚀 Objectives

- Transform raw cricket data into actionable insights
- Practice data cleaning and visualization techniques
- Develop business intelligence and analytical skills
- Create an interactive dashboard for decision-making
